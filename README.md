# Atharve Dahima - Portfolio

Source code for my personal portfolio website, live at **[atharveeee.vercel.app](https://atharveeee.vercel.app)**.

## What's inside

- Home page with an audio intro and GSAP scroll animations
- Projects gallery with 8 case-study pages (drones, LiFi, PCB design, cryogenic CMOS research and more)
- Publications page in a Google Scholar style list
- Lab tour photo essay
- Contact form with direct WhatsApp integration
- Privacy and terms pages

## Stack

- Static HTML, CSS and JavaScript
- GSAP + ScrollSmoother for animations
- Deployed on Vercel with clean URLs

## Running locally

Serve the folder with any static server, for example:

```bash
npx serve .
```

Then open `http://localhost:3000`.

## Structure

```text
├── index.html          home
├── about.html          longer story
├── projects.html       gallery, links into projects/
├── projects/           8 case-study pages
├── publications.html   papers and preprints
├── labtour.html        workshop photo essay
├── contact.html        contact form
└── assets/             css, js, images, audio
```
